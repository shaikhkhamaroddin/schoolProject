This is the side project gave to me from ShimBi Computing Laboratories Private Limited Pune.
It is based on student information which contains crud operation as well as multiple file upload.

Important Features of this project
-Multi file upload (IMAGE,PDF,EXCEL)
-MD5 encrypted Password
-Dependable dropdowns (Country, State, City)
-Jquery plugins (datepicker,ckeditor,fancybox,datatables local and serverside)
-Ajax functionality to enable/disable user account
-Excel file generation
-PDF file generation
-Zip file generation
-Normal search, date range search
-Session storage for login,logout functionality
-Pagination
